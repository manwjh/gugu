# Gugu Media Kit

## One-liner

Gugu is an AI desktop lifeform for macOS: a tiny bird that senses work rhythm, remembers, grows, and speaks at the right moment.

## Short Description

Gugu is a source-first macOS project for an AI desktop lifeform. It is not a chatbot with a skin. It lives on the desktop as a SpriteKit bird, reacts to touch and dragging, senses local work rhythm without recording input content, keeps local memory, grows over time, and uses an OpenAI-compatible model endpoint for low-frequency heartbeats, conversation, and dream-like memory distillation.

## Boundaries To Mention

- Source-first release, not a polished signed app
- macOS 14+ and Swift 5.9+ required
- Requires an OpenAI-compatible model endpoint and API key
- Camera and microphone are optional and off by default
- No video or GIF assets should be used for this launch

## Recommended Order

1. GitHub-owned channels: already complete.
2. X / Twitter: short post with Pages link and optional preview PNG.
3. V2EX after account activation: use `分享创造`.
4. HN Show HN: no image, concise technical positioning.
5. Reddit: post to one subreddit first, wait for response, then adapt for the second.
6. Chinese long-form: publish a polished article after the first wave of feedback.

## Launch Metrics

- Stars
- Website visits if analytics are added later
- Release asset downloads
- Issues and discussions with real feedback
- Number of external users who successfully build and run Gugu

## Blockers

- Browser automation requires the Codex Chrome Extension in the selected Chrome profile.
- Third-party posting requires login state and final confirmation before public submission.
- V2EX requires legal account activation; do not use leaked or sold invitation codes.

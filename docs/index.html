Gugu

An AI desktop lifeform: a tiny bird that senses work rhythm, remembers, grows, and speaks at the right moment.

View on GitHub Open v2.3.0 Release

Not a chatbot with a skin.

Gugu is designed as a desktop creature with a body, rhythm, memory, and boundaries. It is currently for macOS users who are comfortable building from source.

Desktop body

Programmatic SpriteKit bird with movement, blinking, grooming, idle behavior, dragging, throwing, touch reaction, and window perching.

Ambient timing

It does not wait for every prompt. A low-frequency heartbeat lets it speak only when curiosity and timing line up.

Work rhythm

Keyboard and mouse sensing counts activity rhythm, not input content. Focused work freezes model heartbeats.

Local memory

Shared history, skills, events, proposals, audits, and growth state live in local files that can be inspected or deleted.

Growth stages

Daily dream distillation turns recent events into memory and behavior experience, giving the bird continuity over time.

Source-first

macOS 14+, Swift 5.9+, and an OpenAI Chat Completions compatible endpoint are required for this release.

  • OpenAI-compatible model transport for heartbeat, conversation, and memory distillation.
  • Optional camera, wake word, and local TTS are off by default.
  • Tool permissions and capability expansion go through local proposal approval.
  • Release assets are static PNG images only.

Local-first boundaries

Raw sensing stays on the Mac.

No typed content capture Keyboard and mouse sensing counts rhythm only.
Camera and mic are opt-in They are off by default and controlled locally.
No raw audio/video upload The model receives text summaries, not sensory streams.
Inspectable local state Memory, audit logs, config, and proposals are local files.

# 咕咕演示视频脚本

目标长度:20-40 秒。核心目标不是讲完功能,而是让人立刻明白“它像一个活物,不是聊天窗口”。

## 画面结构

### 0-5 秒:第一眼

画面:

- 桌面上有一只咕咕站在底部。
- 鼠标靠近,咕咕轻微反应。

字幕:

```text
咕咕:一个活在 macOS 桌面上的 AI 小生命
```

### 5-12 秒:身体与互动

画面:

- 咕咕走动、眨眼、理毛。
- 拖拽它,松手抛出去,落地后恢复。

字幕:

```text
有身体,会互动,不是聊天机器人套皮
```

### 12-20 秒:工作节奏

画面:

- 切到编辑器或浏览器。
- 咕咕退到一边或安静待着。

字幕:

```text
你专注时,它不打扰,也不调用大模型
```

### 20-30 秒:择机开口

画面:

- 停下输入。
- 咕咕冒出一句短气泡,例如“刚停下来呀?”

字幕:

```text
它会在合适的时机说一句短话
```

### 30-38 秒:隐私与开源

画面:

- 菜单栏开关或 README 隐私段落一闪而过。
- 最后停在 GitHub 页面或 README 顶部。

字幕:

```text
摄像头/麦克风默认关闭。源码已公开。
github.com/manwjh/gugu
```

## 录制建议

- 用浅色或中性色桌面背景,避免咕咕被背景吃掉。
- 不展示 API key、配置文件真实 key、聊天日志隐私内容。
- 视频里少放解释性文字,只保留短字幕。
- 第一版不要超过 40 秒。
- 中文社区可用中文字幕;HN/Reddit/X 使用英文字幕。

## 英文字幕版本

```text
Gugu: an AI desktop lifeform for macOS
It has a body. It is not a chatbot with a skin.
When you focus, it stays quiet and stops model heartbeats.
It speaks only when the timing feels right.
Camera and microphone are off by default. Source is public.
github.com/manwjh/gugu
```
